# ✨ Modern Developer Portfolio

A sleek, responsive, and interactive portfolio website designed to showcase my skills, projects, and professional journey. Built using modern web technologies, this portfolio features a clean user interface with dynamic animations, glassmorphic elements, and a dark/light mode toggle.

## 🎨 Features

- **Responsive Design:** Optimized for all screen sizes, from mobile devices to large desktop monitors.
- **Modern UI/UX:** Clean typography, curated color palettes, smooth transitions, and subtle hover effects.
- **Project Showcase:** A dynamic grid displaying project cards with links to live demos and GitHub repositories.
- **Skills Section:** Visual indicators/badges showcasing technical proficiency across frontend, backend, and tools.
- **Interactive Contact Form:** A user-friendly form with client-side validation to receive inquiries directly.
- **Performance Optimized:** Fast load times with optimized assets, clean code structure, and search engine optimization (SEO) best practices.

---

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic markup)
- **Styling:** CSS3 (Custom variables, Flexbox/Grid layouts, Glassmorphism effects)
- **Interactivity:** JavaScript (ES6+ for dynamic rendering, active navigation, and form validation)
- **Animations:** CSS transitions and keyframes for smooth visual flow

## 📦 Getting Started

Follow these steps to run the portfolio locally on your machine.

### Prerequisites
To view the site, you only need a modern web browser. For development, a local server tool (like VS Code's Live Server extension or Node.js) is recommended.

### Local Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd portfolio
   ```
3. **Open the project:**
   - If using VS Code, right-click `index.html` and select **Open with Live Server**.
   - Or, if you're using a tool like Vite/Node, install dependencies and run:
     ```bash
     npm install
     npm run dev
     ```

## ⚙️ Customization

To personalize this portfolio with your own information:

1. **Personal Details:** Open `index.html` and replace all placeholder text (such as `[Your Name]`, bio, and social links) with your own info.
2. **About Me & Skills:** Update the profile description and modify the skill list to match your technical profile.
3. **Projects:** Locate the projects section and replace the details, image sources, and links with your own work.
4. **Assets:** Place your profile picture and project screenshots in an `assets/` or `images/` directory.
